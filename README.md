# Study-Algorithm
-------------

## 폴더 구성
### 1. algorithm-skill-note
기본적인 알고리즘(BFS, sorting 등)들을 "파이썬"으로 작성   

### 2. baekjoon
백준 문제풀이 작성.
- 문제 번호는 반드시 다섯자리   
예) 2345번 문제 -> 02345   
- 파일 이름 형식은 "문제번호"-(문제 타입)-(나중에 다시 볼지 여부)
